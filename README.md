### Usage

```
npm install
npm server
open loaclhost:3000/webpack-dev-server/app
```

Now edit 'app/index.js' & 'app/style.css'.  
Your changes will appear without refresh the browser.

### Dependencies
* webpack
* webpack-dev-server
* babel-loader
